 69        """
 70        Click a Radio Button by supplying either the text of the Radio button or the value of the Radio button. The

 72

105        """
106        Determine if the Radio button identified by the text or value is currently selected.
107
108        :param button_text: The text of the Radio button for which you would like to check the selection state. If

117            return self._get_input(button_text=button_text, button_value=button_value).find().is_selected()
118        except TimeoutException:
119            return False

189        Obtain the wrapper in use for the button of the Radio Group as a ComponentPiece. This wrapper contains the
190        input, icon, and label for the 'button'.
191

193            have the same exact text the first match will be selected. Takes precedence over button_value.
194        :param button_value: The value of the Radio button from which you'd like the wrapper.
195
196        :returns: ComponentPiece
197        """

201        wrapper = self._radio_wrapper_coll.get(attr_text)
202        if not wrapper:

207                parent_locator_list=self.locator_list)
208            self._radio_wrapper_coll[attr_text] = wrapper
209        return wrapper
